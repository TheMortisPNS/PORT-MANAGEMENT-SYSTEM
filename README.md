## 📌 Περιγραφή Έργου

Το **Port Management System** είναι μια web εφαρμογή διαχείρισης λιμενικών αφίξεων και αναχωρήσεων πλοίων.  
Στόχος του έργου είναι η οργάνωση και η παρακολούθηση των κινήσεων πλοίων σε ένα λιμάνι, η αποθήκευση βασικών στοιχείων κάθε άφιξης, η διαχείριση του status των πλοίων και η καταγραφή οικονομικών δεδομένων τύπου **PDA (Proforma Disbursement Account)**.

Η εφαρμογή αναπτύχθηκε στο πλαίσιο πανεπιστημιακής εργασίας και συνδυάζει:

- Web development με **PHP**
- Αποθήκευση δεδομένων σε **MySQL / MariaDB**
- Διεπαφή χρήστη με **HTML / CSS / Bootstrap / JavaScript**
- Δικτυακή προσομοίωση με **Cisco Packet Tracer**
- Υποστήριξη ανάπτυξης με χρήση εργαλείων **AI**

## 🎯 Σενάριο Εφαρμογής

Σε ένα λιμάνι υπάρχουν καθημερινά πλοία που:

- αναμένονται να φτάσουν,
- βρίσκονται ήδη στο λιμάνι,
- ή έχουν αναχωρήσει.

Η διαχείριση αυτών των πληροφοριών πρέπει να γίνεται με οργανωμένο τρόπο, ώστε ο χειριστής του συστήματος να μπορεί:

- να καταχωρεί νέα πλοία,
- να επεξεργάζεται υπάρχουσες εγγραφές,
- να παρακολουθεί την κατάσταση κάθε πλοίου,
- να βλέπει προγραμματισμένες αφίξεις και αναχωρήσεις σε ημερολόγιο,
- να ενημερώνει το real-time status,
- να υπολογίζει και να αποθηκεύει κόστη PDA,
- να πραγματοποιεί αναζητήσεις και να εξάγει χρήσιμα στατιστικά.

Η εφαρμογή λειτουργεί ως ένα απλό πληροφοριακό σύστημα διαχείρισης λιμένα με έμφαση στην οργάνωση των δεδομένων, στην ευχρηστία και στην οπτική παρακολούθηση της κίνησης των πλοίων.

## 🧱 Αρχιτεκτονική Εφαρμογής

Η εφαρμογή ακολουθεί αρχιτεκτονική **client-server**.

### Δομή αρχιτεκτονικής

1. **Client πλευρά**
   - Ο χρήστης αλληλεπιδρά μέσω browser.
   - Η διεπαφή περιλαμβάνει φόρμες, πίνακες, ημερολόγιο, στατιστικά και status board.

2. **Application layer**
   - Η επιχειρησιακή λογική υλοποιείται σε αρχεία PHP.
   - Η PHP διαχειρίζεται τα αιτήματα εισαγωγής, αναζήτησης, επεξεργασίας, διαγραφής και ενημέρωσης status.

3. **Data layer**
   - Τα δεδομένα αποθηκεύονται στη βάση **MySQL/MariaDB**.
   - Ο βασικός πίνακας της εφαρμογής είναι ο `arrivals`.

### Ροή λειτουργίας

- Ο χρήστης ανοίγει την εφαρμογή από τον browser.
- Συμπληρώνει φόρμα για νέα άφιξη ή επεξεργάζεται υπάρχουσα εγγραφή.
- Η PHP λαμβάνει τα δεδομένα από τη φόρμα.
- Τα δεδομένα αποθηκεύονται ή ενημερώνονται στη MySQL.
- Η εφαρμογή εμφανίζει τα αποτελέσματα σε λίστα, ημερολόγιο, PDA προβολή, στατιστικά και status board.

### Βασικά backend αρχεία

| Αρχείο | Ρόλος |
|---|---|
| `src/index.php` | Αρχική σελίδα με λίστα όλων των εγγραφών και συνοπτική προβολή κινήσεων |
| `src/add_ship.php` | Φόρμα εισαγωγής νέου πλοίου / ταξιδιού |
| `src/insert.php` | Εισαγωγή νέας εγγραφής στη βάση |
| `src/edit_ship.php` | Επεξεργασία υπαρχόντων στοιχείων πλοίου |
| `src/delete.php` | Διαγραφή εγγραφής |
| `src/search.php` | Αναζήτηση πλοίων με λέξεις-κλειδιά |
| `src/calendar.php` | Ημερολογιακή προβολή αφίξεων και αναχωρήσεων |
| `src/port_status.php` | Real-time status board για τα πλοία |
| `src/update_status.php` | AJAX endpoint για αλλαγή κατάστασης και αποθήκευση σημειώσεων |
| `src/get_ship.php` | Επιστρέφει στοιχεία πλοίου σε μορφή JSON |
| `src/pda.php` | Διαχείριση και προβολή PDA εξόδων |
| `src/statistics.php` | Παρουσίαση στατιστικών στοιχείων |
| `src/helpers.php` | Βοηθητικές συναρτήσεις για ημερομηνίες, ασφάλεια εμφάνισης και υπολογισμούς |
| `src/db_connect.php` | Σύνδεση με βάση δεδομένων |
